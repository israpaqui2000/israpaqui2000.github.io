Project analysis, Visual design: describe using around 200 words
(minimum) the main decisions you have made for the visual design.


To choose my visual style, I opted for a muted color palette, creating strong contrast where color appears but doesn't dominate. To make it easier to understand, I've divided my style into five main points.

- Color Palette: Neutral tones combined with accents of violet and blue to convey creativity and professionalism. I didn't want to use overly bright colors because for a CV, it's best not to tire the reader's eyes with very flashy colors.

- Typography: I used legible sans-serif fonts with good contrast, highlighting titles and skill percentages. This way, I separate and encourage the reader to focus on certain aspects, such as highlighting important skills in bold.

- Iconography: To avoid overwhelming the page with text, I opted to display certain information, such as professional skills with percentages and technical skills with progress bars.

- Animations: I added animations and some transitions to make the page feel more dynamic and less static. I've also made it so that certain elements change color when the user hovers over them.

In short, the goal of my design has been to create a coherent, attractive, and professional visual experience.
